# OwOMacro

OwOMacro is a 4 key macropad with a rotary encoder.

## Features:
- Rotary encoder
- 4 Keys

Here should be everything you need to make this hackpad

- 4x Cherry MX Switches
- 4x Keycaps of your choice
- 1x PEC12R Rotary Encoder
- 1x XIAO RP2040
- 1x Case
